# Pilot KPI Baseline and Cadence

## KPI Set
- Completion rate: submitted applications / started applications.
- Stage drop-off: percent leaving at each form stage.
- Re-upload rate: applications with at least one `needs_reupload` document.
- Review turnaround: median time from `submitted` to first admin status update.

## Baseline Window
- Baseline period: first 7 pilot days.
- Refresh cadence: daily snapshot, weekly trend review.

## Ownership
- Parent completion + drop-off: product/portal owner.
- Re-upload + turnaround: admissions operations owner.

## Reporting Rhythm
- Daily: 10-minute checkpoint with prior-day numbers and blockers.
- Weekly: summary trend note with top 3 risks and next actions.
