# Claude Code Handoff Instructions

Use the shared handoff workflow at the end of substantial coding sessions. If there has been meaningful work, generate or refresh a Markdown handoff in the OS temp directory for the next agent. If the day was quiet or purely administrative, skip it. If the session is active but not ready to close, leave a brief reminder to decide whether to invoke handoff after the sprint.

Keep the handoff concise, redact sensitive details, and include suggested skills for the next agent. Prefer the same handoff artifact format used by Codex and Antigravity so the next platform can continue without re-reading the full thread.
